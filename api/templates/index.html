API Peliculas

Esta API le permite obtener datos de peliculas de la base de datos relacional y la base de datos no relacional

GET generos

Para Obtener todas los Generos de peliculas

Endpoint:

http://127.0.0.1:8000/api/generos/

GET ONE genero

Para Obtener uno de los Generos de peliculas

Endpoint:

http://127.0.0.1:8000/api/generos/1

GET usuarios

Para Obtener todos los usuarios de peliculas

Endpoint:

http://127.0.0.1:8000/api/usuarios/

GET ONE usuario

Para Obtener uno de los usuarios de peliculas

Endpoint:

http://127.0.0.1:8000/api/usuarios/1/

GET peliculas

Para Obtener todas las peliculas

Endpoint:

http://127.0.0.1:8000/api/peliculas/

GET ONE pelicula

Para Obtener una de las peliculas

Endpoint:

http://127.0.0.1:8000/api/peliculas/1/

GET peliculas por genero

Para Obtener todas las peliculas de un genero

Endpoint:

http://127.0.0.1:8000/api/peliculas/genero/1

GET todas las opiniones

Para Obtener todas las opiniones de peliculas

Endpoint:

http://127.0.0.1:8000/api/opinions/

GET todas las peliculas de una opinion

Para filtrar las peliculas por una opinion

Endpoint:

http://127.0.0.1:8000/api/peliculas/opinion/1

GET una pelicula aleatoria

Para obtener una pelicula al azar

Endpoint:

http://127.0.0.1:8000/api/peliculas/random/
